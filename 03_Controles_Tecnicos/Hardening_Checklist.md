# Checklist de Hardening (Windows 10/11)

Este procedimiento se aplica a todos los equipos nuevos antes de entregarlos al usuario.

- [ ] **BIOS:** Contraseña de BIOS configurada. Boot desde USB deshabilitado.
- [ ] **Cifrado:** BitLocker activado en unidad C:. Clave guardada en AD/Gestor.
- [ ] **Cuentas:** Usuario local "Invitado" deshabilitado. Usuario diario es "Estándar" (no Admin).
- [ ] **Red:** Firewall de Windows activado (Perfil Dominio/Público/Privado).
- [ ] **Software:** Desinstalar bloatware (juegos, demos).
- [ ] **Actualizaciones:** Windows Update configurado en automático.
- [ ] **Antivirus:** Windows Defender actualizado y protección en tiempo real activa.
